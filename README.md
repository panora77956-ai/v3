# veo3

## Features

### Voice Prosody & Speaking Style Controls

The application now includes comprehensive voice prosody and speaking style controls to ensure professional, consistent tone across all scenes.

**Key Features:**
- 6 pre-configured speaking style presets (Professional, Conversational, Storytelling, Educational, Enthusiastic, Calm & Relaxed)
- Fine-tune controls for speaking rate, pitch, and expressiveness
- Apply settings globally to all scenes for consistency
- Supports both Google TTS (with SSML) and ElevenLabs

For detailed documentation, see [docs/VOICE_PROSODY_FEATURE.md](docs/VOICE_PROSODY_FEATURE.md)
